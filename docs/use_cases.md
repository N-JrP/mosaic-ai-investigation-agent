# Use Cases

Mosaic will be tested with real document and business data.

## Document use cases

| Use case | What it proves |
|---|---|
| Extract vendor name from receipts | The system can find key document fields |
| Extract receipt date | The system can read date information from documents |
| Extract total amount | The system can identify important financial values |
| Find receipts with missing fields | The system can flag documents needing review |
| Search receipts by keyword | The system can retrieve relevant documents |

## Business intelligence use cases

| Use case | What it proves |
|---|---|
| Find high-revenue product categories | Mosaic can analyze real revenue from Olist orders |
| Find product categories with high revenue and weak reviews | Mosaic can combine sales and customer satisfaction signals |
| Analyze delivery delays by state | Mosaic can support operations analysis |
| Show monthly revenue trend | Mosaic can create time-based business insights |
| Find common payment methods | Mosaic can analyze customer payment behavior |

## Combined use cases

| Use case | What it proves |
|---|---|
| Move from documents to structured insights | The system connects document intelligence with analytics |
| Ask business questions in plain English | The system supports non-technical users |
| Show SQL-backed answers | The system gives transparent, verifiable results |
| Explain business meaning | The system does more than show raw data |
| Suggest next actions | The system supports decision-making |
